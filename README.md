# Ninja-Game

Ninja-Game: A 2D pixel-art platformer built with Python and Pygame.

## 🚀 Overview

This project was developed during my internship at **[LiveTech-India]**. The goal of the project was to learn about Python.

* **Language:** Python
* **Libraries:** Pygame
* **Tools:** VS Code, etc.

## ⚙️ Installation & Setup

1.  **Install dependencies:**
    ```bash
    pip install pygame-ce
    ```
2.  **Run the project:**
    ```bash
    python UI.py
    ```
